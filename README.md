# N-Star-pattern

In this task, you have to print n stars \* vertically and n horizontally.<br />

Input Format:<br />
The first line of input contains n.<br /><br />

Output Format:<br />
print the stars in horizontal and vertical direction.

Example 1<br />
Input<br />
3<br /><br />
Output<br />
/_\*\*<br />
/_<br />
/_<br />
/_<br />

Exaplantion<br />

We print the 3 stars in horizontal and 3 stars vertical direction.<br /><br />

Example 2<br />
Input<br />

5<br /><br />
Output<br />
/**\***<br />
/_<br />
/_<br />
/_<br />
/_<br />
/\*<br />
Exaplantion<br />

We print the 5 stars in horizontal and 5 stars vertical direction.<br /><br />

Constraints<br />
1<= n <= 100
